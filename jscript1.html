변수
하나의 값을 저장하기 위해 확보한 메모리 공간 자체 또는 공간의 이름

변수 이름 = 식별자
식별자는 값이 아닌 메모리 주소를 기억
변수 선언: var, let(변수), const(상수)
변수 선언 후 undefined 할당
호이스팅: 변수 선언문은 평가단계에서 미리 실행
var, let 재할당 가능

표현식과 문
표현식: 값으로 평가될 수 있는 모든 문
문: 프로그램을 구성하는 기본 단위, 최소 실행 단위

값: 표현식이 평가되어 생성된 결과
리터럴: 값 자체
완료값: 항상 undefined

데이터 타입
값의 종류 7가지

자바는 동적타입 언어이다
=> 유연성은 높지만 신뢰성은 떨어진다

원시타입: number, string, boolean, undefined, null, symbol
객체타입: object, function
